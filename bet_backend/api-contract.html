NBA Box Scores API

API Contract · Base URL: http://localhost:8080

GET /boxscores

List player box scores in the database (paginated), ordered by game date descending, then game ID, then athlete ID.

Query Parameters

NameTypeRequiredDescription
page_idintegerNoPage number (min: 1, default: 1)
page_sizeintegerNoItems per page (1–100, default: 10)

Response

200 OK — Array of PlayerBoxscore

Data Models

PlayerBoxscore

Full box score record. All fields may be null.

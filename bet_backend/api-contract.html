NBA Analytics API

API Contract · Base URL: http://localhost:8080

All list endpoints are paginated. Response shape: {"data": [...], "page_id": 1, "page_size": 10}

GET /boxscores

List player box scores (paginated), ordered by game date descending, game ID, athlete ID.

Query Parameters

Name Type Required Description
page_id integer No Page number (min: 1, default: 1)
page_size integer No Items per page (1–100, default: 10)

Response

200 OK — data: array of PlayerBoxscore

GET /play-by-play

List play-by-play events (paginated), ordered by game date descending, game ID, game play number.

Query Parameters

Name Type Required Description
page_id integer No Page number (min: 1, default: 1)
page_size integer No Items per page (1–100, default: 10)

Response

200 OK — data: array of PlayByPlay

GET /team-boxscores

List team box scores (paginated), ordered by game date descending, game ID, team ID.

Query Parameters

Name Type Required Description
page_id integer No Page number (min: 1, default: 1)
page_size integer No Items per page (1–100, default: 10)

Response

200 OK — data: array of TeamBoxscore

GET /schedules

List game schedules (paginated), ordered by game date descending, game ID.

Query Parameters

Name Type Required Description
page_id integer No Page number (min: 1, default: 1)
page_size integer No Items per page (1–100, default: 10)

Response

200 OK — data: array of Schedule

500 Internal Server Error

Database or server error.
